# Changelog

All notable changes will be documented here. NeuMan follows semantic versioning after the public compatibility surface is ratified; pre-1.0 releases may change experimental interfaces only with explicit migration notes.

## Unreleased

- Added the initial durable architecture and SPEC-00 through SPEC-22.
- Added the Rust core/CLI, local SQLite ledger and CAS, Git/Rojo controls, Roblox publication guard, Studio bridge/plugin, optional self-hosted Hub, GitHub App boundary, and Tauri/React desktop.
- Added Roblox public-client OAuth with S256 PKCE, exact loopback redirect, OIDC validation, fail-closed OS-vault persistence, mandatory refresh rotation, and bounded no-redirect transport.
- Added accepted native-art capture/fan-out with explicit bases, context-bound transfers, large authenticated loopback downloads, dirty-cell protection, and atomic Studio apply receipts.
- Added signed native-execution contracts and crash-safe operator Open Cloud execution state.
- Added protected GitHub release workflow sources, native signing/notarization/update/provenance contracts, and fail-closed release preflight.

No production-ready or official binary release has been declared.
